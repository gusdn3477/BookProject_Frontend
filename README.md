## 온라인 서점 프로젝트
프로젝트 이름 : 3조 책방  
프로젝트 개요 : MSA 방식을 활용하여 온라인 서점 웹 사이트를 제작 및 배포한다.  
팀원 : 권진희, 김영모, 박현우   
프로젝트 기간 : 2주  
<br/>

사용하는 언어: HTML5, CSS3, Javascript  
프론트엔드 : React.js  
백엔드 : Spring Boot, Spring Cloud Eureka, AWS RDS(Mariadb), AWS EC2, Docker   
협업 툴: github, freedcamp(일정관리)

### 사용법
- Frontend의 경우, npm install 후 사용해 주세요.
